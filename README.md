This Android application, developed in **Kotlin using Android Studio**, is designed to keep users updated with real-time **notifications about campus or organizational events, meetings, and workshops**. The app allows any authorized user to create an event by entering its name and scheduled time. Once the event is saved, an **instant notification is broadcasted to all users** who have the app installed, ensuring everyone stays informed and punctual. Ideal for **college campuses, offices, or community groups**, this app simplifies communication and eliminates the hassle of manual reminders, providing a **centralized and efficient event notification system** for all members.
